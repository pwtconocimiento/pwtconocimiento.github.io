Ampliar la RAM de tu PC

Profr. Juan Carlos Cuello

¿QUÉ ES LA RAM?

 
La RAM es una memoria virtual que se utiliza para que nuestra pc pueda guardar temporalmente los programas que se utilizan, así como los archivos que estemos trabajando (audio, video, texto, etc.).

La RAM es una tarjeta electrónica que se añade en un ranura dentro de nuestra PC, es donde se almacenan los datos que utiliza nuestra computadora pero de manera temporal, RAM significa Random Access Memory (memoria de acceso aleatorio) y es por eso que es tan importante que cuando se compra una Pc, dentro de las características se menciona cuanto de RAM tiene.

Ampliar la memoria RAM

 
Ampliar la memoria RAM de nuestra PC nos están complicado como aparenta ser, solo necesitamos:
  1. Un desarmador, pero para eso debe fijarse qué tipo de tornillos tiene su CPU

2. Una vez que hayas quitado la tapa, el siguiente paso sería quitar la tarjeta RAM a cambiar y colocar la nueva tarjeta, dependiendo del modelo de CPU, puede que tenga 2, 3 o 4 ranuras.

3. Lo siguiente que nos queda es volver a colocar la tapa y atornillar

Como nota importante debe verificar como inserta la memoria evitando forzarla, ya que lo estarías insertando mal.

Bueno como podrán leer no es cosa complicada poder ampliar nuestra memoria RAM, solo hace falta unos pocos pasos pero, si no están muy seguros entonces recomiendo que lo lleven con un técnico calificado.
